 Social Media Analytics

Objective
To analyze social media engagement metrics and identify high-performing content.

 Dataset
CSV file containing post-level data such as likes, comments, shares, and reach.

 Analysis Performed
- Platform-wise engagement analysis
- Content type performance comparison
- Engagement correlation analysis
- Visualization of insights

Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

 Outcome
Identified that visual content generates higher engagement across platforms.
