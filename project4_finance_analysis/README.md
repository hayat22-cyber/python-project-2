 Finance Market Analysis

 Objective
To analyze stock market data and evaluate price movements and returns.

Dataset
CSV file containing stock price data including open, close, and volume.

 Analysis Performed
- Daily return calculation using percentage change
- Trend visualization
- Stock performance comparison
- Correlation analysis

 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

 Outcome
Identified stock performance trends and volatility patterns.
