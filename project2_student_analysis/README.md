Student Performance Analysis

 Objective
To analyze student academic performance and study the impact of attendance on scores.

Dataset
CSV file containing student marks, attendance percentage, and personal details.

 Analysis Performed
- Pass and fail analysis
- Subject-wise performance comparison
- Attendance vs marks correlation
- Data visualization

Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

 Outcome
Found that higher attendance generally leads to better academic performance.
