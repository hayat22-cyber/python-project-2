Weather Data Analysis

 Objective
To analyze weather data and identify temperature and rainfall patterns.
 Dataset
CSV file containing daily weather records such as temperature, rainfall, humidity, and wind speed.

Analysis Performed
- Temperature trend analysis
- Rainfall distribution analysis
- Seasonal pattern identification
- Correlation analysis

 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
 Outcome
Identified seasonal weather patterns and trends useful for planning and forecasting.
